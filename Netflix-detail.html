Project Information

  • Category: Web Development
  • Client: Evangadi Tech
  • Project date: 19 June, 2025
  • Project URL: https://nafy-10.github.io/Netflix-Clone-2025/

Project Overview

This project is a fully functional replica of the Netflix streaming platform, showcasing my expertise in web development. Built with React for the frontend and Node.js for the backend, it includes user authentication, a dynamic movie database, and a responsive design. Users can browse, search, and stream content, with personalized recommendations based on their viewing habits. This project highlights my skills in full-stack development and user experience design.

Key Features

Fully Responsive Design

Optimized for all devices from mobile to desktop with adaptive layouts

Dynamic Movie Database

A comprehensive library of movies and TV shows, complete with detailed information, ratings, and genres, enabling users to explore a wide range of content.

Search and Filter Options

Advanced search functionality and filtering options that allow users to easily find specific titles or browse by categories.

Personalized Recommendations

An algorithm that suggests content based on users' viewing history and preferences, enhancing the user experience by providing tailored suggestions.
